# Kotndeks Browser (K2K)

**Kotndeks Browser** - это анонимный локальный браузер/редактор на базе K2K, позволяющий создавать и расшифровывать безопасные ссылки внутри проекта без использования серверов. Проект ориентирован на приватность и полную автономность.

---

## Функции

* Создание **Kotndeks-ссылок** с шифрованием текста.
* Расшифровка полученных ссылок.
* Поддержка **кастомных доменов** для ссылок.
* Вкладки для одновременной работы с несколькими ссылками.
* Минималистичный интерфейс в стиле Chrome или даркнета.
* Локальный редактор текста с возможностью шифрования и генерации ссылок.
* Полностью автономная работа без подключения к интернету (кроме обмена ссылками вручную).

---

## Установка и использование

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Thepshnk/kotndeks
```

2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).

3. Для создания ссылки:

   * Введите текст.
   * Укажите домен (по желанию).
   * Нажмите "Создать ссылку".

4. Для расшифровки:

   * Вставьте Kotndeks-ссылку или payload.
   * Нажмите "Прочитать".

---

## Пример использования

* Ввод текста: `Привет, мир!`
* Домен: `Site.krutoy`
* Генерация ссылки: `kotndeks://Site.krutoy/JIOJE3OIJIOjei32nhjkrh3u4yiehw`
* Расшифровка этой ссылки вернет исходный текст.
